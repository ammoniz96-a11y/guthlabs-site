Request a Figure

This page describes a planned feature. It is not built yet. When it ships, you will be able to ask for the mind your work needs, and every requested figure will pass the same gates as the catalog before it ships.

How It Will Work

  1. You name the figure and the works it should be built from.
  2. We source a public-domain edition and scrub it.
  3. The mind is baked, gated, and graded, and published with its receipts like every other entry in the catalog.
  4. If it cannot pass the gates, it does not ship, and we say why.

Who Gets Priority

Professor PIN holders (see Education) will be actively prompted to request figures in their own field, because an expert's request is also the best test we can get. A PIN is access, never endorsement.
