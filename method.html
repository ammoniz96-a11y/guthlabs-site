Method

Guth Figures / method

The claim this project makes is narrow and checkable: every figure is footnoted and falsifiable. We do not claim the system is beyond error. We claim that every assertion carries a receipt you can go and check yourself, and that when a check fails, the failure is recorded instead of papered over.

The verbatim law

No text in any figure's corpus is generated. A corpus is built only from a named public-domain edition (a Project Gutenberg or Wikisource text, cited by number on each figure's detail page). The engine fetches that source, verifies the title and author of what it actually received against what it expected, and refuses to ingest anything that does not match - a mismatch is logged as a skip, never patched over. The text is then cut into verbatim chunks, passages of the source as published, and the chunk count you see on each card is counted from those files on disk.

Where to check: the fetch-verification and refuse-on-mismatch logic is in engine/figure-engine.mjs (the FETCH-MISMATCH path); per-figure results, chunk counts, and word counts are appended to engine/LEDGER.md after every run.

The grades

Every figure carries an A-D voice-fidelity grade, always visible on its card - never buried. The grade answers one question: how close is this corpus to the figure's own voice? The rubric, quoted from the grading code:

A = rich first-person corpus, voice is genuinely theirs
B = strong but filtered (dictated, third-person, or editor-curated)
C = secondhand reconstruction
D = mostly synthetic or thin sources

A grade is a human judgement, recorded with written reasons. The engine does not invent grades; it records the letter and basis a human supplied, attaches the corpus and test evidence, and writes the file. A grade is not complete until a person signs it - until then the site shows it plainly as, for example, "grade A (unsigned)". Fictional figures get the same treatment, with an extra honesty requirement: where a character's voice lives inside someone else's narration (Ahab inside Ishmael's story, Gatsby inside Nick's), the grade basis says so.

Where to check: the rubric and signing rule are in engine/grade.mjs; each figure's written grade lives in its own GRADE.md.

Receipts

Every layer keeps its paperwork:

  • Corpus receipts. Each figure's detail page names the exact edition and license basis. The engine's ledger records what was fetched, whether it verified, how many chunks and words resulted, and what artifacts were written.
  • Test receipts. Each corpus is run against a bait suite - trap questions designed to catch a figure claiming knowledge outside its corpus. The checks are string and vector matching, not a language model grading itself; results are written to the figure's BAIT_RESULTS.md with a plain pass rate.
  • Image receipts. Every portrait image on the gallery has its own source record (busts/<slug>.SOURCE.md) naming the Wikimedia Commons file page, the license template actually observed on that page, and the retrieval date. Where a likeness is doubtful, the record says so - the Seneca bust's record notes that modern scholarship questions the identification.

Falsifiable means falsifiable by you: pick any quote-to-be, any chunk count, any license line, and trace it to the named source. If it does not trace, that is a defect, and reporting it is doing us a favor.

What this site is not

There is no chat here yet. The conversation engine (retrieval plus quote verification, phase 2) is under construction; this site catalogs the verified corpora it will draw from. Nothing on these pages is a transcript of a figure speaking.
